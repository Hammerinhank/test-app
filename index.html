🅿️ Historique des stationnements
✍️ Saisir des coordonnées à la main
Colle ou tape les coordonnées sous la forme 48.2580226, 7.4589611 (les parenthèses, points-virgules ou espaces multiples sont acceptés). Le point sert de séparateur décimal. Le stationnement est ajouté à l'historique comme s'il avait été relevé sur place, avec recherche automatique de l'adresse.
⚙️ Synchronisation GitHub (optionnelle, chiffrée)
Stockage local toujours actif : l'historique est pérennisé dans le navigateur (localStorage) et fonctionne même sans GitHub. La synchro GitHub s'ajoute par-dessus, en redondance : token et phrase restent sur cet appareil et ne sont jamais écrits dans le code ni envoyés sur GitHub ; seul l'historique chiffré part dans parkings.json.
💾 Sauvegardes de trajets
Chaque sauvegarde crée un fichier chiffré dans le dossier trajets/ de ton dépôt, nommé Trajet AAAA-MM-JJ HHhMM <nom>.json. Le rechargement ajoute les stationnements à l'existant, sans rien écraser, en respectant l'ordre chronologique. Le bouton d'effacement ne s'active que si le contenu actuel a été sauvegardé.
🅿️ Détection automatique du stationnement (expérimental)
Quand c'est activé, l'appli relève ta position environ toutes les 2 minutes (position grossière, économe en batterie), estime ta vitesse, et distingue les moments où tu es en voiture des moments où tu ne l'es pas. Après une phase de conduite suivie de 30 minutes sans rouler, un stationnement est créé automatiquement à l'endroit où la voiture s'est arrêtée.

Limite importante (iPhone) : une page web ne tourne pas en arrière-plan. La détection ne fonctionne donc que tant que cette appli reste ouverte à l'écran ; elle se met en pause si tu verrouilles le téléphone ou changes d'application.
🧪 Simuler le raccourci Apple
Ce bouton fait tout en une fois : il publie cette page dans ton dépôt sous le nom localisation.html (il le crée s'il n'existe pas, via ton token GitHub), s'assure que GitHub Pages est activé, puis construit l'URL de test ?parking=… avec ta position actuelle — celle que ton raccourci devra ouvrir. Un token GitHub (section Synchronisation) est requis pour la publication.
